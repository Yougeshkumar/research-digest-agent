# 🧠 Research Digest Agent

An autonomous AI-powered research assistant that ingests multiple sources, extracts key insights, removes redundancy, and generates a structured, evidence-backed research digest.

---

## 🚀 Features

* 🔍 Fetches and cleans content from multiple web sources
* 🧠 Extracts meaningful claims using NLP heuristics
* 🧩 Groups similar insights using TF-IDF + cosine similarity
* 📊 Generates structured research summaries
* 🎨 Interactive HTML dashboard
* 🌙 Dark / ☀️ Light mode toggle
* 🔎 Search functionality
* 🎯 Theme-based filtering

---

## 🏗️ Tech Stack

* Python
* BeautifulSoup
* Scikit-learn
* HTML, CSS, JavaScript

---

## ▶️ Run Locally

```bash
git clone https://github.com/yougeshkumar22/research-digest-agent.git
cd research-digest-agent

pip install -r requirements.txt
python src/main.py
```

👉 Then open:
outputs/digest.html
